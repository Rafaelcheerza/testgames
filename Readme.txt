Este es mi primer juego subido a una plataforma.

El juego propone ser al estilo "Fruit Ninja".

Se lanzarán 20 piedras que deberás cliquear para romperlas y ganar puntuación.

Es un juego desarrollado con las instrucciones de un curso de Unity.

Posee máximo unas 8 horas de desarrollo.

La intención de subirlo a la github es para ensayar y entender cómo funciona la plataforma.
